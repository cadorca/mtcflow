MTCFLow

MTCFlow is framework that allows the creation of model-based solutions, using a Graphical DSL you can design, develop, test and deploy any Model Transformation Chain (MTC). The tool integrates technologies like : EPSILON, XTEXT, ATL, OCL, QVTO, ACCELEO and VELOCITY in the Eclipse IDE.
With MTCFlow you can easily create and maintain MDE solutions, like Code Generators.

Using MTC Flow you can put the Model Driven Engineer in practice. These are the key features that it offers:

  • A Graphical editor for defining any MTC
  • Transformation Technology Interoperability
  • Validation Technology Interoperability
  • Alternative execution paths for MTCs
  • Integrated Development Environment on top of the Eclipse IDE
  • Support for testing while you develop an MTC
  • Stand-alone execution of MTCs
  • Extensible to integrate any transformation technology
